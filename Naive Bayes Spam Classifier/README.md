# Naive Bayes Spam Classifier

This program predicts if a message is spam or not.

## What it does
- Counts words in spam and not spam
- Uses probability to guess spam
- Prints result in console

## Real-life examples
1. Movie review sentiment: positive or negative.  
2. Classifying news articles into categories like Sports, Politics, or Entertainment.  

## What we learn
- How AI uses statistics
- What probability means
- How computer makes simple decisions
